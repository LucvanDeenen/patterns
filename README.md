# Design Patterns Training (C# / .NET 10)

This repository contains small example implementations of common design patterns in C# using .NET 10.

## Prerequisites

- .NET SDK 10.0+

## Run an example

From the repository root, run one of the projects e.g.:

```bash
dotnet run --project ./AbstractFactoryPattern/AbstractFactoryPattern.csproj
dotnet run --project ./AdapterPattern/AdapterPattern.csproj
dotnet run --project ./SingletonPattern/SingletonPattern.csproj
...
```

## Notes

This is a learning/training repository focused on simple, readable pattern demonstrations.
